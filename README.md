# Shelf: Writing Notes

A lightweight and efficient Git-based repository for personal notes, research, and technical documentation.

## Overview

This repository provides a minimalistic structure to store and manage notes using Markdown. By using Git version control, every change is tracked, allowing for easy collaboration, history review, and synchronization across devices.

## Project Structure

- **`notes/`**: The primary directory for all Markdown (`.md`) notes.
- **`LICENSE`**: The MIT license governing this repository.

## Getting Started

1.  **Creation**: Add your markdown files directly into the `notes/` directory.
2.  **Version Control**: Commit your changes to keep a history of your writing:
    ```bash
    git add .
    git commit -m "Add new note on [Topic]"
    ```
